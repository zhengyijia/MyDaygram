# MyDaygram
Imitate the app called "Daygram"("素记" in Chinese).<br>
##实现了：
* 不同item的显示<br>
* 星期天的红色显示<br>
* 长按删除<br>
* 编辑界面添加时间点<br>
* 月份的切换和年份的切换<br>
* 切换listView的另一种显示模式<br>
* 数据的文件保存<br>
* 密码设置功能<br>
